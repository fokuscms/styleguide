Forms

There are two different types of forms: The simple basic form and the inline form.
Both of the have the same markup and only differ in a single CSS class: .inline

Basic form

The basic form just puts one thing at a time.

Some additional text to explain the controls limitation such as file-types.

Inline Form

An inline form tries to put as much elements in a row as possible. If you want to create a mixture of vertical and horizontal elements use the basic form combined with a grid.

Radio buttons and Checkboxes need additional classes to put them in a row
